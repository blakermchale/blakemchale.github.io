Autonomous Drone - In Progress

This fall I started my own personal project developing an autonomous drone. For my drone I am using a pixhawk 4 with a raspberry pi 3 B+. This drone will use the MAVLINK capability of the pixhawk to communicate between the two systems. MAVLINK works by sending commands to the pixhawk 4. Currently, I am trying to get the two to communicate with each other however it is taking time to find out the format of the messages used for the pixhawk 4 software. I have written a program in opencv using python that tracks the largest face in view which I plan on having it translate face translation to motor movement.
